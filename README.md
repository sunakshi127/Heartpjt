# Heart Disease Prediction System

## Project Description

This project predicts the likelihood of heart disease using Machine Learning algorithms and Flask web deployment.

## Features

- Heart disease prediction
- Interactive web interface
- Real-time prediction
- Flask deployment

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS

## Installation

```bash
git clone YOUR_REPOSITORY_LINK
```

```bash
pip install -r requirements.txt
```

```bash
python app.py
```

## Machine Learning Models

- Logistic Regression
- KNN
- Random Forest
- SVM

## Deployment

Deployed using Render.
